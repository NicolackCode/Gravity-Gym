# AI Core Rules - Lien vers \_brain/

Tu dois, au début de chaque session, te référer aux fichiers situés dans `_brain/` :

- `_brain/architecture.md` (Choix de stack, topologie).
- `_brain/workflow.md` (Méthode de travail, conventions).

Rappels critiques (Définis par le USER) :

1. Processus "Étape par Étape". Présenter la prochaine phase et attendre un "GO" humain.
2. PAS de code tronqué (pas de "// ... code ici ...").
3. PAS de destruction silencieuse de code.
4. Proactivité : utiliser le Terminal, le navigateur, et valider les étapes soi-même.
5. On ne chaîne pas de commandes avec `&&` dans le terminal.
